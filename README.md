# 🎵 BeatBox Studio

BeatBox Studio is a modern and interactive browser-based soundboard built using **HTML, CSS, and JavaScript**. It allows users to play different sounds with a single click, control the playback volume, and stop all currently playing sounds. The application features a clean, responsive interface and stores the user's preferred volume using LocalStorage.

---

## 🚀 Features

- 🎵 12 unique sound buttons
- 🔊 Adjustable volume slider
- ⏹️ Stop All Sounds button
- 💾 Volume preference saved using LocalStorage
- 📱 Fully responsive design
- ✨ Modern glassmorphism user interface
- ⚡ Smooth hover and click animations

---

## ▶️ How to Run

1. Clone or download this repository.
2. Ensure all sound files are placed inside the `sounds` folder.
3. Open `index.html` in your browser.
4. Click any sound button to play audio.
5. Adjust the volume using the slider or stop all sounds using the **Stop All Sounds** button.
